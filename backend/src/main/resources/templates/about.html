About the App
This app is a smart coding platform designed for students and teachers. Students can submit their code for problems, and the app automatically analyzes it to estimate time and space complexity, detect potential code smells, and provide feedback on coding quality.

Teachers can easily review submissions, track progress, and offer guidance — making the learning and evaluation process more efficient and transparent.

It's a bridge between learning, practice, and constructive code review, helping students write cleaner and more efficient code.
